# 📋 Daily Job Alert Dashboard
**Author: Partha Pattanayak**

A Python desktop application that aggregates Manual Testing and
Playwright JavaScript QA job listings from 38+ job boards.

# Job Dashboard UI

## Features
- 🔵 Manual Testing jobs
- 🟣 Playwright / JavaScript automation jobs
- Filter by skill or title
- Save favourite listings
- Direct links to 38+ job boards
- Refresh timestamp

## Requirements
- Python 3.6+
- tkinter (built into Python — no install needed)

## How to Run
```bash
python job_alert_dashboard.py
```

## Job Boards Covered
LinkedIn, Indeed, Naukri, Glassdoor, Wellfound, WeWorkRemotely,
RemoteOK, CutShort, Instahyre, Hirist, Dice, TestDevJobs,
NoFluffJobs, Himalayas, BuiltIn, Remotive, and more...
